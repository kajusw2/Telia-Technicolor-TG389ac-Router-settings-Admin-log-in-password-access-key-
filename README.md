# Telia-Technicolor-TG389ac-Router-settings-Admin-log-in-password-access-key-
How to Access Telia Technicolor TG389ac Router Settings (Using Access Key)
This guide is for Telia Technicolor TG389ac routers (older firmware around 2015–2016).
It shows how to access the router settings without contacting Telia.
Only do this on your own router.
What you need:
Physical access to the router
The sticker on the router
A device connected to the router (Wi-Fi or Ethernet)
Step 1: Find the Access Key
Look at the label on the back or bottom of the router. You will see:
Wi-Fi name (SSID)
QR code
Access Key
Important:
The Access Key is NOT the Wi-Fi password
The Access Key is used to log in to the router via a web browser
Step 2: Open the router configuration page
Open a web browser and go to: 192.168.1.1
(or use your device’s default gateway if this is different)
Step 3: Log in to the router
When the login page appears, use:
Username: admin
Password: Access Key (from the router sticker)
Step 4: Manage settings
After logging in, you can:
View and change Wi-Fi settings
Change the Wi-Fi password
Reset the router from the web interface
Important notes:
This router has NO physical reset pinhole
All resets and configuration are done through the web interface
No need to contact Telia if you have the Access Key
Summary:
Username is admin
Password is the Access Key
Access Key is different from the Wi-Fi password
Everything is done in the browser

WARNING:
YOU MIGHT NEED TO RESET THE PASSWORD AND THIS CAN DEFAULT THE SETTINGS LIKE A FACTORY DATA RESET.
